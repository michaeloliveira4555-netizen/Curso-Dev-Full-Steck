                                Comando Explicação simples
git --version                                           Verifica se o Git está instalado.
git config --global user.name "Seu Nome"                Define o nome do autor dos commits.

git config --global user.email "seuemail@email.com"     Define o email do autor dos commits.

mkdir projeto                                           Cria uma pasta para o projeto.
cd projeto                                              Entra na pasta criada.
git init                                                Inicializa o repositório Git local.
git status                                              Mostra arquivos novos ou modificados.

git add .                                               Prepara todos os arquivos para commit.

git commit -m "mensagem"                                Salva uma versão local do projeto.
git branch -M main                                      Renomeia a branch principal para main.

git remote add origin URL                               Conecta ao repositório remoto no GitHub.

git remote -v                                           Lista os remotos configurados.
git push -u origin main                                 Envia o projeto para o GitHub.
git log                                                 Mostra histórico completo de commits.

git log --oneline                                       Mostra histórico resumido.
git branch                                              Lista branches locais.
git remote set-url origin URL                           Atualiza a URL do remoto.
git remote remove origin                                Remove o remoto configurado.
